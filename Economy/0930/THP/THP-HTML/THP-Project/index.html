THPプロジェクト一覧

### Project 00 - ドルペッグ国救済プラン

GCC等のドルペッグ維持を円建て決済で橋渡しし、人道回廊を同時起動する即応パッケージ（BCH×JPYC×RORO）。

ドルペッグを維持するための金融調整と人道支援ルートの標準化に関する提案資料。

### Project 01 - 日本救国プラン

決済・物流遮断や大規模流入時でも、生活（食・医・電・治安）を止めないための実務マニュアル群。

国内向けの危機対応マニュアルと訓練ドキュメントを整理し、平時から運用できる形に整備。

### Project 02 - ライフラインインフラ構築

物流遮断時でも食料・医薬・電力・通信を確保するための時限インフラ圏の設計。

  • 資料１：ライフライン・インフラ構築圏に関する細則
  • 資料２：T-5→T+90 行動計画
  • 資料３：ライフライン・インフラ構築圏の設置及び運用に関する費用見積もり（清書版）
  • 資料４：米国債市場における需給逼迫と決済機能不全リスクに関する分析報告（清書版）

### THP-DEATH回避策

初動10兆円（円建て）を一括前払し、対外交渉と人道支援を止めない資金動線の標準手順。

国際交渉と人道支援を同時に維持するための財政・外交アクションリスト。
